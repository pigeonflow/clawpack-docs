# clawpack init

Create a `manifest.json` in the current directory. Use this when you have an existing agent workspace and want to publish it.

## Usage

```bash
clawpack init
```

## Options

None.

## What It Does

Creates a minimal `manifest.json` with prompts for name, version, and description. If you're starting from scratch, use [`clawpack create`](/cli/create) instead — it generates the full workspace.
